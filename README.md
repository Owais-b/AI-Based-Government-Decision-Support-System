# AI-Driven Government Decision Intelligence Platform

**Decision Intelligence Engine** for government officers using NYC 311 complaints.

An AI-powered decision support platform designed to assist government authorities in making data-driven, transparent, and efficient policy decisions. The system leverages Machine Learning, Data Analytics, and Artificial Intelligence to analyze public datasets, identify trends, generate predictive insights, and recommend evidence-based actions.

---

## 📌 Project Overview

Governments deal with enormous volumes of structured and unstructured data collected from multiple departments. Traditional decision-making processes are often time-consuming and susceptible to human bias.

This project introduces an AI-powered Decision Support System that analyzes historical and real-time governmental data to generate actionable insights, helping policymakers make faster, smarter, and more transparent decisions.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Owais-b/AI-Based-Government-Decision-Support-System.git
```

### Navigate to Project

```bash
cd AI-Based-Government-Decision-Support-System
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

Run: `streamlit run app.py`
