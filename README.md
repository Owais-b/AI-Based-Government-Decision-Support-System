# AI-Driven Government Decision Intelligence Platform

**Decision Intelligence Engine** for government officers using NYC 311 complaints.

Features:
- NLP understanding + Sentiment + Urgency
- Crisis early warning & trend prediction
- Decision Recommendation Engine
- Smart Resource Allocation Optimizer
- Policy Impact Simulator
- AI Copilot (Chat Assistant)
- Interactive Maps + Explainable AI
- Auto PDF Reports

Run: `streamlit run app.py`
